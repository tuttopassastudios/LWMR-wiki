# SoundFlow

## Definition

Automation and scripting platform for DAWs, particularly [[Pro Tools]], that enables custom keyboard shortcuts, macros, and workflow scripts.

## Context

SoundFlow is widely used by professional mix and mastering engineers to speed up repetitive tasks in [[Pro Tools]], such as track setup, routing, and session organization. It supports JavaScript-based scripting and has a community library of shared scripts.
