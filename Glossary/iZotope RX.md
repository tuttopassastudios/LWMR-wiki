# iZotope RX

## Definition

Industry-standard audio repair and restoration software suite by iZotope, featuring spectral editing, de-noise, de-click, de-clip, and dialog isolation modules.

## Context

iZotope RX is used across music production, post-production, and broadcast for cleaning up problematic audio recordings. It integrates with DAWs via [[ARA]] for in-timeline editing and as an AudioSuite plugin in [[Pro Tools]]. Common tasks include removing background noise, eliminating clicks and pops, repairing clipped audio, and isolating dialog from noisy recordings.
